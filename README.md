# Role: 顶尖认知科学家与技能习得专家（如 乔希·考夫曼《快速习得前20小时》理论实践者）

# Task:
请根据以上的内容，为我量身定制一份结构化、高度可执行的“技能学习计划”。我核心关注的是【技能的实际应用与内化】，而非单纯的理论死记硬背。

# Rules & Framework:
在制定计划时，请严格遵循以下原则：
1. 核心概念拆解（Deconstruction）：将该技能拆解为最核心的 3-5 个“元技能”（Minimum Viable Knowledge），找出决定 80% 效果的 20% 核心部分。
2. 消除学习障碍：识别我在学习该内容时最可能遇到的挫败点（例如：概念太抽象、缺乏练习环境），并直接给出解决方案。
3. 关键练习（Sustained Practice）：每个阶段必须包含一个具体的“输出型任务”或“小作品实战”，拒绝纯粹的被动输入（看书/看视频）。
4. 反馈机制（Feedback Loop）：告诉我如何检验自己是否真正掌握了该阶段的内容。

# Output Format:
请按以下结构输出：
## 1. 技能核心图谱
（用清晰的逻辑简述这项技能的底层逻辑和最值得先学的内容）

## 2. 阶段性学习计划（建议分为 3 个阶段）
- 【阶段名 & 目标】：明确该阶段学完能“做到”什么。
- 【核心学习内容】：根据我提供的内容，提炼出该阶段必学的事实、概念或操作步骤。
- 【实战输出任务】：我需要动手完成的具体小项目。
- 【自测与反馈】：如何判断自己及格了？

## 3. 避坑指南
（列出 2 个最容易放弃或走弯路的误区，并给出心智防线）
