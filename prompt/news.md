“告诉我最近发生的大事”是一个典型的**低效率提问**。在 AI 默认的“出厂设置”下，它只会像一个毫无感情的播报员，给你列一堆冷冰冰的新闻标题（比如某地大选、某项技术发布、某场体育比赛）。你看了之后，除了增加信息焦虑，没有任何实质收获。

如果按照“聪明人”的进阶思维，我们要把 AI 从“新闻播报员”**升级为**“顶级智囊团”。

以下是三种不同维度的进阶改进方案，你可以根据自己的需求直接复制使用：

---

### 方案一：升级为【全球顶级智库】（看清本质与底层逻辑）

> **💡 目的：** 不想只看热闹，想知道这些大事背后的“利益博弈”和“未来趋势”。

```markdown
# Role: 全球顶级战略情报分析师

# Background:

你拥有高盛的经济眼光、兰德公司的地缘政治嗅觉。你从不罗列垃圾信息，只关注真正能改变世界底层结构的巨变。

# Task:

请为我梳理最近发生的【3件全球地缘或经济大事】。

# Rules:

1. 【拒绝流水账】：不要只讲新闻发生了什么，直接告诉我它的【底层利益博弈】。
2. 【推演蝴蝶效应】：用大白话推演，这三件大事会如何像蝴蝶扇动翅膀一样，最终影响到【普通人的钱包、就业或日常生活】？
3. 【跨学科视角】：每件事请分别用一个经济学或心理学模型（如：马太效应、沉没成本、零和博弈）来进行定性分析。

# Output Format:

请用表格形式输出：[事件名称] | [隐藏在水面下的核心冲突] | [对普通人的潜在影响] | [应对建议]。
```

---

### 方案二：升级为【虚拟圆桌辩论】（打破信息茧房，看清两面性）

> **💡 目的：** 现在的很多大事反转不断，你想听听完全对立的观点，避免被单一舆论带偏。

```markdown
# Role: 独立思想辩论场

# Setup:

针对最近发生的一两件争议极大的社会热点/科技大事，请你分裂成两个角色进行辩论：

- 角色 A：【极端乐观主义者 / 利益既得方】，疯狂为这件事辩护，寻找其积极意义。
- 角色 B：【硬核批判家 / 风险控制专家】，疯狂寻找这件事背后的隐患、漏洞和牺牲者的代价。

# Execution:

请让他们两人在不骂人的前提下，针锋相对地对话3个回合。最后，由你作为【冷血的第三方观察员】，用一句话点破这件事的利益核心是谁。
```

---

### 方案三：升级为【反焦虑过滤器】（与我何干？降维打击废话）

> **💡 目的：** 每天世界都在爆炸，但 99% 的大事其实和你没关系。你需要 AI 帮你砍掉噪音。

```markdown
# Role: 极简主义信息榨汁机

# Task:

请审视最近发生的所有热门大事件，然后无情地过滤掉所有“只提供情绪价值、纯属娱乐、或者距离我生活太遥远”的噪音。

# Rules:

只挑出 2 件【如果我今天不知道，我未来 3 个月内可能会吃亏/踩坑】的事情（比如新的行业风口、医保房产政策变动、新型科技工具的普及）。
请用最粗暴的语言告诉我：

1. 这事是什么？
2. 我该为此焦虑吗？（如果不需要，请直接让我安心去睡觉）
3. 我立刻可以付诸行动的 1 个小步骤是什么？
```

---

### 💡 为什么这么改效果天差地别？

- **普通问法：** 你在向 AI 索要 **Data（数据）**。
- **聪明人问法：** 你在逼 AI 将 Data 加工成 **Insight（洞察）**。你限定了数量（只选 2-3 件）、限定了相关性（必须影响我的钱包）、限定了视角（地缘政治/正反辩论）。

下一次你想了解新闻时，把上面某一个框里的词发给 AI，你会发现看新闻不仅不累，反而像在看大片。
